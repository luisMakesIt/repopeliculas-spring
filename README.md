## Getting started

```bash
git clone https://github.com/luisMakesIt/repopeliculas-spring.git
cd repopeliculas-spring
```

### Install dependencies

```bash
./mvnw test  # or mvn test when no wrapper exists
```

### Run locally

```bash
./mvnw spring-boot:run
```
